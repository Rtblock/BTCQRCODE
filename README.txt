Bitcore Address Generator & QR code Generator.
1. Open command prompt in "Bitcore-Demo-2" folder.
2. Execute the following command - 
	>node app.js
3. Following text appears on the command prompt screen - 
	Started Server at port : 8088
	It means the server is started.
4. Open a browser and type in the following URL
	http://localhost:8088/
5. Click on the button "Click me to generate address".
6. Address and the corresponding QR code is generated on the next page that follows.
